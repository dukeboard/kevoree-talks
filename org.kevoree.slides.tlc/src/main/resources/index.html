TAA : Technique d'Architecture logicielle Avancée

TLC : Technique logicielle pour le Cloud Computing

Olivier Barais

Triskell Team / INRIA / IRISA , 2012

Salut

TAA

  • Objectifs
    • Maîtriser les concepts et les outils pour construire une architecture logicielle complexe
      • Principe d'architecure logicielle à base de composants
      • Rappel sur les outils de mapping objet relationnels
      • Spring, EJB3, OSGi, GWT, Maven, Play
    • Projet de mise en oeuvre pratique pour chacune des technologies étudiées en cours

TLC

  • Objectifs
    • Maîtriser les concepts et les outils du cloud computing et comprendre les enjeux pour la construction d'applications scalables
      • Concept du cloud computing
      • Stockage de données elastiques
      • La vision google (introduction à GAE)
      • La vision Typesafe (présentation de la pile logicielle de typesafe)
      • La vision Amazon (S3, EC2, SQL)
      • Adaptation multi-niveaux à l'aide de Kevoree
    • Projet de mise en oeuvre pratique pour chacune des technologies étudiées en cours
