Java annotation API

Kevoree annotation API aims at decorate your Java code with simple annotations to declare your own Kevoree type definition.

Annotation are parsed @Compile time to generate a Kevoree model fragement and generate Scala Kevoree usefull classes for runtime deployment.

Compilation process

In a nushell Kevoree compile time process is composed by these steps :

  • Parse Kevoree annotation with javax.annotation standard processor
    • TOTO
    • TUTU
  • Generate a lib.kev file in KEV-INF directory
  • Compile your Java or Scala classes
  • Generate & compile Kevoree Scala runtime classes
  • Package everything in a jar, embedded KEV-INF/lib.kev file
TOTO

Maven plugin

The easiest way to use Kevoree annotation tools is to use Maven POM project description. We provide a maven plugin dedicated to extend your compilation process with Kevoree steps. In addition Kevoree API and Framework are also available throught our maven repositories.

Maven plugin (2)

Add the following dependencies to your Maven Project


    
        org.kevoree.tools
        org.kevoree.tools.javase.framework
        ${kevoree.version}
    
    
        org.kevoree.tools
        org.kevoree.tools.annotation.api
        ${kevoree.version}

Maven plugin (3)

Add the following plugin to your Maven Project. NodeTypeNames can take the following values : JavaSENode, AndroidNode, ArduinoNode
